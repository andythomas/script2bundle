## How to use
Simply run the script to generate an example file.

Available options:
1. executable: The filename of the script to be bundled.
2. icon: The filename of the icon file (e.g. png) to be used
3. destination: Where to place the .app file
  - executable: The same directory as the executable files
  - system: /Applications
  - user: ~/Applications
4. name: The desired name of the applications
